# IPL-Data-Analysis
Comprehensive IPL Data Analysis using Python, Pandas, Matplotlib, and Seaborn to uncover team performance, player statistics, match trends, and business insights through exploratory data analysis.

#  IPL Data Analysis using Python

##  Project Overview

This project performs an Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset using Python. The analysis aims to uncover valuable insights into team performances, player achievements, venue statistics, match trends, and factors influencing match outcomes through data cleaning, visualization, and business-driven analysis.

The project uses two related datasets (`matches.csv` and `deliveries.csv`) to answer real-world business questions by applying Pandas, data merging techniques, and data visualization.

---

#  Problem Statement

The Indian Premier League generates a vast amount of match and ball-by-ball data every season. Analyzing this data can help identify team strengths, player performances, match-winning factors, venue trends, and scoring patterns.

The objective of this project is to perform exploratory data analysis (EDA) on historical IPL data and extract meaningful business insights using Python.

---

#  Objectives

- Understand the structure and quality of the IPL datasets.
- Perform data cleaning and preprocessing.
- Analyze team and player performances.
- Study batting and bowling trends.

#  Key Findings

- Mumbai Indians have won the highest number of IPL matches.
- Gujarat Titans recorded the highest overall win percentage.
- Winning the toss has only a marginal influence on match outcomes.
- Virat Kohli is the highest run scorer in IPL history.
- Yuzvendra Chahal has taken the most wickets.
- Eden Gardens has hosted the highest number of IPL matches.
- Mumbai is the city that has hosted the most IPL matches.
- Teams scoring above 180 runs have a significantly higher probability of winning.
- Lucknow Super Giants recorded the highest average runs in winning matches.
- Exceptional batting performances frequently contribute to Player of the Match awards.
